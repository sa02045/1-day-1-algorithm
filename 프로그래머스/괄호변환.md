# 괄호변환

1. 균형잡힌 괄호 문자열 개념
   - '(' ')'의 갯수가 서로 같으면
2. 올바른 괄호 문자열 개념

   - 갯수도 같고, 짝도 맞으면

3. 문제에서 구하는것: 주어진 균형을 -> 올바른으로 변환한 결과

4. 문제풀이 단계:

   1. 빈문자열은 빈문자열을 반환
   2. u,v로 분리하는데, u는 분리할수 없는 균형괄호, v는 빈문자열 가능

      - u가 올바른괄호라면 v에 대해 다시 수행

   3. u가 올바른 괄호가 아니라면
      - 빈 문자열에 '('를 붙이고
      - v에 문자열을 이어붙인다
